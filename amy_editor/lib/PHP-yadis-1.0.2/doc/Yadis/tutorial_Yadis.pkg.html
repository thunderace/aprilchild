PHP Yadis API

JanRain, Inc.
openid@janrain.com

This is an implementation of the Yadis service discovery protocol. This package contains a Yadis discovery implementation and an XRDS parsing implementation.

Using This Package

To install this package, copy the Services/ directory in this package to a directory in your PHP include path. Alternatively, modify your PHP include path to include the directory that contains Services/. Any applications that need this package will then be able to use it.

For notes on how to begin developing with this package, see the Services_Yadis_Yadis class documentation.
